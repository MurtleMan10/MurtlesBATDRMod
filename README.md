# MurtlesBATDRMod

Mod created by Murtle
Current Version: v1.0.1
Mod for: Bendy and the Dark Revival created by Joey Drew Studios

HOTKEYS:

Q = Teleport Forward
9 = Teleport Up
0 = Teleport Down
U = Freeze/Unfreeze Character # Use outside of cutscenes
K = Unlock Character # Use in cutscenes
H = Heals Player
J = Hide UI
L = Show UI
